l.a.a.BigString(_Argument) : class documentation

Part of lib.amp.arguments View In Hierarchy

Known subclasses: lib.amp.arguments.BigUnicode

Encodes an arbitrarily large chunk of data by dynamically adding key/value pairs to the AMP Command packet. This is not a standard encoding scheme.

This is little more than a hack, and services no purpose save convenience for some simple applications.

This is taken from http://amp-protocol.net/Types/BigString/
